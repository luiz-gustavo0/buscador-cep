## Rodando o projeto localmente

Clone este repositório

```bash
  git clone https://github.com/luiz-gustavo0/buscador-cep.git
```

Vá até a pasta do projeto

```bash
  cd buscador-cep
```

Instale as dependências

```bash
  npm install ou yarn install
```
